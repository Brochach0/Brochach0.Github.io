Written Response for GUI

The Java GUI project was a bit less challenging than the ones that came before it. One of the problems that we had encountered during this program was the font type not working. When we would switch our font type to Times new Roman it would show and error that there was no PLAIN text field and would not be able to use a java supported font. This issue was fixes by adding another JLabel above it with empty text. Another issue that we had confronted was our tic tac toe code would save when it was reopened. If someone was playing the game and X won, close the game and it would start up displaying X won again. We fixed this issue by searching up a portion that added a variable and made the code completely exit when it was closed.
